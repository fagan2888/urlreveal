Using the API

Simply send a GET request with a 'url' variable to www.urlreveal.com/api, e.g.:
http://www.urlreveal.com/api?url=http://bit.ly/h8vhq0

Possible responses are:

  • The destination URL: E.g. http://www.urlreveal.com
  • 301: The URL redirects infinitely
  • 403: The request was forbidden by the target server
  • 404: The page was not found
  • 410: The page is no longer available
  • 500: The URL was invalid or resulted in another error
  • 503: The page's server is unavailable

The URL must be UTF-8 encoded if it contains non-ASCII characters.
